    Part 1 - Single Machine Distributed Sorting (60%)

    • Create a single machine distributed system that sorts a file of numbers where there is one number per line.
    • You must implement your solution using threads.
    • You can use any language you want.
    • Your solution must be merge sort based (https://en.wikipedia.org/wiki/Merge_sort).
    • Solution must be able to run on your Ubuntu server virtual machine.

    Part 2 - Multi-machine Distributed Sorting (40%)

    • Implement a machine distributed version of part 1.
    • You must sort using 3 or more machines.
